# Projects-C

Collection of small C projects and learning exercises. 

How to build a project:

- Open the project folder and run:

```
make
```

Or to build and run (when provided):

```
make run
```

Projects:

- `atm` — ATM demo using a binary file for client balances.
- `calculator` — simple interactive calculator.
- `guessthenumber` — number guessing game.
- `sorting` — student-record sorting example (moved from `ordenacao`).
- `binarysearch` — iterative binary-search implementation (ready to use).
- `image-compression` — translated image/P3-PPM helper (renamed from Portuguese `compressao`).
- `temperature` — temperature converter (conversion functions fixed).
- `data-structures/` — collection of implementations: `stack`, `linkedstack`, `queue`, `linkedlist`, `bintree`.
